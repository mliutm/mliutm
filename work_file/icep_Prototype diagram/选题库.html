列表页面规则：

1. 页面初始化：

       1）进入该页面时在提示创建新任务成功后执行‘零申报’按钮功能，成功新建当期任务后不再执行前述逻辑（【境外员工】本页面也加上该功能）

       2）零申报图标：默认隐藏，如果【本期收入】中 ’申报类型 = 0‘则显示，否则隐藏


2. 查询区：



3. 按钮区：             

        获取税局数据：单击该按钮规则如下：

            境内员工，境外员工-居民：

             a.

             b. 后台启动RPA拉取累计数保存到本系统：

             c.本月从未单击本按钮则打开【税款计算统计】页面时弹出对话框提示：请更新累计数！，点‘确定’则启动RPA从扣缴端抓取上表的累计数并保存



            境外员工-非居民：

             a. 执行原保存功能

             b. 后台启动RPA拉取‘已缴税额’保存到本系统：

             c. 如果本月从未单击本按钮则打开【税款计算统计】页面时弹出对话框提示：请更新已缴纳额！，点‘确定’则启动RPA从扣缴端抓取‘已缴纳额’并保存


           零申报：【境外员工-居民/非居民】本页面也加上该功能， 规则如下：

                  1）如果‘当前所属月份’的【税表审批】状态 = 待审核 则本按钮置灰，否则可用

                  2）该功能会联动【税款计算统计/税表审批/一键报税】页面

                  3）点该按钮弹出提示框：确定当期为零申报？点‘确定’则使当前的‘税款所属月份’对应的所有记录‘本期收入’ = 0.00，申报类型 = 0，而后自动保存所有记录

                 4）【境外员工-非居民】：点该按钮时自动清空‘适用公式 ，境内工作天数，境外工作天数，境内支付(人民币)，境外支付(人民币)‘

’


           保存：点按钮如果有一条记录的本字段值 不等于 0则使‘申报类型’ = 1（【境外员工-居民】本页面也加上该功能）

        


4. 字段取值规则：

           申报类型：增加该字段，隐藏，值为：0（表示：零申报），1（表示：正常申报），默认值=1

           本期收入：手工修改或批量导入或引入基本工资/复制上月公司时，如果有一条记录的本字段值 不等于 0则使‘申报类型’ = 1

列表页面规则：

1. 页面初始化：

       1）进入该页面时在提示创建新任务成功后执行‘零申报’按钮功能，成功新建当期任务后不再执行前述逻辑（【境外员工】本页面也加上该功能）

       2）零申报图标：默认隐藏，如果【本期收入】中 ’申报类型 = 0‘则显示，否则隐藏


2. 查询区：



3. 按钮区：             

        获取税局数据：单击该按钮规则如下：

            境内员工，境外员工-居民：

             a.

             b. 后台启动RPA拉取累计数保存到本系统：

             c.本月从未单击本按钮则打开【税款计算统计】页面时弹出对话框提示：请更新累计数！，点‘确定’则启动RPA从扣缴端抓取上表的累计数并保存



            境外员工-非居民：

             a. 执行原保存功能

             b. 后台启动RPA拉取‘已缴税额’保存到本系统：

             c. 如果本月从未单击本按钮则打开【税款计算统计】页面时弹出对话框提示：请更新已缴纳额！，点‘确定’则启动RPA从扣缴端抓取‘已缴纳额’并保存


           零申报：【境外员工-居民/非居民】本页面也加上该功能， 规则如下：

                  1）如果‘当前所属月份’的【税表审批】状态 = 待审核 则本按钮置灰，否则可用

                  2）该功能会联动【税款计算统计/税表审批/一键报税】页面

                  3）点该按钮弹出提示框：确定当期为零申报？点‘确定’则使当前的‘税款所属月份’对应的所有记录‘本期收入’ = 0.00，申报类型 = 0，而后自动保存所有记录

                 4）【境外员工-非居民】：点该按钮时自动清空‘适用公式 ，境内工作天数，境外工作天数，境内支付(人民币)，境外支付(人民币)‘

’


           保存：点按钮如果有一条记录的本字段值 不等于 0则使‘申报类型’ = 1（【境外员工-居民】本页面也加上该功能）

        


4. 字段取值规则：

           申报类型：增加该字段，隐藏，值为：0（表示：零申报），1（表示：正常申报），默认值=1

           本期收入：手工修改或批量导入或引入基本工资/复制上月公司时，如果有一条记录的本字段值 不等于 0则使‘申报类型’ = 1
