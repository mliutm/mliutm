01-处理器问题

发表于2023-12-20|更新于2023-12-21|面试题华为OD100分
|字数总计:2k|阅读时长:10分钟|阅读量:

题⽬描述

某公司研发了⼀款⾼性能AI处理器。每台物理设备具备8颗AI处理器，编号分别为0、1、2、3、4、

5、6、7。

编号0-3的处理器处于同⼀个链路中，编号4-7的处理器处于另外⼀个链路中，不通链路中的处理器

不能通信。

如下图所示。现给定服务器可⽤的处理器编号数组array，以及任务申请的处理器数量num，找出符

合下列亲和性调度原则的芯⽚组合。

如果不存在符合要求的组合，则返回空列表。

亲和性调度原则：

-如果申请处理器个数为1，则选择同⼀链路，剩余可⽤的处理器数量为1个的最佳，其次是剩余3个

的为次佳，然后是剩余2个，最后是剩余4个。

-如果申请处理器个数为2，则选择同⼀链路剩余可⽤的处理器数量2个的为最佳，其次是剩余4个，

最后是剩余3个。

-如果申请处理器个数为4，则必须选择同⼀链路剩余可⽤的处理器数量为4个。

-如果申请处理器个数为8，则申请节点所有8个处理器。

提示：

  1. 任务申请的处理器数量只能是1、2、4、8。

  2. 编号0-3的处理器处于⼀个链路，编号4-7的处理器处于另外⼀个链路。

  3. 处理器编号唯⼀，且不存在相同编号处理器。

输⼊描述

输⼊包含可⽤的处理器编号数组array，以及任务申请的处理器数量num两个部分。

第⼀⾏为array，第⼆⾏为num。例如：

[0, 1, 4, 5, 6, 7]

1

表示当前编号为0、1、4、5、6、7的处理器可⽤。任务申请1个处理器。

  • 0 <= array.length <= 8
  • 0 <= array[i] <= 7
  • num in [1, 2, 4, 8]

输出描述

输出为组合列表，当array=[0，1，4，5，6，7]，num=1 时，输出为[[0], [1]]。

⽤例

输入[0, 1, 4, 5, 6, 7]
1
输出[[0], [1]]
说明根据第⼀条亲和性调度原则，在剩余两个处理器的链路（0, 1, 2, 3）中选择处理器。由于只有0和1可⽤，则返回任意⼀颗处理器即可。
输入[0, 1, 4, 5, 6, 7]
4
输出[[4, 5, 6, 7]]
说明根据第三条亲和性调度原则，必须选择同⼀链路剩余可⽤的处理器数量为4个的环

题⽬解析

⽤例中，链路link1=[0,1]，链路link2=[4,5,6,7]

现在要选1个处理器，则需要按照亲和性调度原则

如果申请处理器个数为1，则选择同⼀链路，剩余可⽤的处理器数量为1个的最佳，其次是剩余

3个的为次佳，然后是剩余2个，最后是剩余4个。

最佳的是，找剩余可⽤1个处理器的链路，发现没有，link1剩余可⽤2，link2剩余可⽤4

其次的是，找剩余可⽤3个处理器的链路，发现没有

再次的是，找剩余可⽤2个处理器的链路，link1符合要求，即从0和1处理器中任选⼀个，有两种选

择，可以使⽤dfs找对应组合
